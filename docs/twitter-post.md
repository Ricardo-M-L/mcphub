I built MCP Hub — the npm for MCP servers 🚀

One command to search, install, and manage MCP servers:

$ mcphub search filesystem
$ mcphub install <server>
$ mcphub doctor  ← diagnoses broken servers!

Auto-configures Claude Code, Claude Desktop & Cursor.

Built in Go. Single binary. Zero deps.
